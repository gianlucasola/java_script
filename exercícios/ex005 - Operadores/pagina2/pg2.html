Auto-atribuição e Incremento em JavaScript

var n = 3
n = n = 4


Inicialmente, definiu-se a VAR chamada n com o valor 3.
Na segunda linha de código, houve uma auto-atribuição:

A variável n passou a valer o seu valor inicial (3) somado a 4, agora vale 7!

É possível simplificar uma auto-atribuição:

  • [ n = n + 4 ] pode ser escrito n += 4
  • [ n = n - 5 ] pode ser escrito n -= 5
  • [ n = n * 6 ] pode ser escrito n *= 6
  • [ n = n / 7 ] pode ser escrito n /= 7
  • [ n = n % 8 ] pode ser escrito n %= 8
  • [ n = n ** 9 ] pode ser escrito n **= 9

Além da auto-atribuição, podemos usar símbolos para representar decremento ou incremento:

- Supondo que definimos: var n = 4

[ n++ ]*, os dois + significam que a variável n será acrescida a um, passando ao valor de 5;

[ n-- ]*, os dois - significam que a variável n será diminuída em um, passando ao valor de 4 (já que o n++ havia passado seu valor para cinco).

*Com os sinais após o nome da variável (n++ ou n--), a mudança no valor não é exibida imediatamente, tendo que "chamar" a variável para notar a alteração.
