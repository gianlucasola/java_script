Operadores Lógicos em JavaScript

Nesta ordem de precedência:

  • ! significa NEGAÇÃO;
  • && significa CONJUNÇÃO;
  • || significa DISJUNÇÃO.

Negação:

  • ! true (não é verdadeiro)
  • ! false (não é falso)

Conjunção (precisa que as duas condições sejam verdadeiras):

  • true && true = TRUE
  • true && false = FALSE
  • false && true = FALSE
  • false && false = FALSE

Disjunção (basta que uma condição seja verdadeira):

  • true || true = TRUE
  • true || false = TRUE
  • false || true = TRUE
  • false || false = FALSE
