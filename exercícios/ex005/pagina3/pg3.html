Operadores Relacionais em JavaScript

  • > significa "MAIOR QUE";
  • < significa "MENOR QUE";
  • >= significa "MAIOR OU IGUAL QUE";
  • <= significa "MENOR OU IGUAL QUE";
  • == significa "IGUAL A";
  • === significa "IDÊNTICO A";
  • != significa "DESIGUAL (DIFERENTE)".

Relacionais são como comparações, e retornam true ou false.

  • 3 > 7 = false
  • 4 < 8 = true
  • 5 >= 6 = false
  • 6 <= 9 = true
  • 7 == '7' = true [porque '==' aceita "comparação" entre number e string]
  • 8 === '8' = false ['===' precisa que seja o mesmo valor E o mesmo tipo de dado]
  • 9 != 10 = true [porque nove é desigual a dez]
